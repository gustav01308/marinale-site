# Mecânica Marinale — Site

Site institucional da [Mecânica Marinale](https://marinale.com.br), oficina mecânica em Florianópolis com mais de 35 anos de experiência.

Construído em **Next.js 14 (App Router)**, **TypeScript** e **Tailwind CSS**.

## Como rodar

```bash
npm install
npm run dev
```

Abra http://localhost:3000.

## Build de produção

```bash
npm run build
npm run start
```
